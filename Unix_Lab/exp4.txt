User Management Commands

cmd :
whoami -> Displays the current logged-in username
Example: whoami

who -> Shows details of users currently logged into the system
Example: who , who -u , who -b

w -> Displays logged-in users and their current activities
Example: w , w username

sudo -> Executes command with superuser (root) privileges
Example: sudo apt update , sudo useradd john , sudo cat /etc/shadow

su -> Switches to another user account (starts shell as that user)
Example: su , su root , su - username

exit -> Exits the current shell/session
Example: exit , exit 0

passwd -> Sets or changes user password
Example: passwd , passwd username

groupadd -> Creates a new group in the system
Example: sudo groupadd developers , sudo groupadd -g 1001 admin

groups -> Displays groups the user belongs to
Example: groups , groups username

usermod -> Modifies user account (e.g., add to group, lock account)
Example: sudo usermod -aG developers john , sudo usermod -L john

groupmod -> Modifies existing group (e.g., change group name)
Example: sudo groupmod -n newname oldname

groupdel -> Deletes a group from the system
Example: sudo groupdel developers

useradd -> Creates a new user account
Example: sudo useradd john , sudo useradd -m john , sudo useradd -g developers john

userdel -> Deletes a user account (with option to remove home directory)
Example: sudo userdel john , sudo userdel -r john

chown -> Changes file owner and/or group
Example: sudo chown john file.txt , sudo chown john:developers file.txt , sudo chown -R john: /home/john

chage -> Displays or modifies user password expiry settings
Example: sudo chage -l john , sudo chage -M 30 john , sudo chage -E 2025-12-31 john

chfn -> Changes user information (name, phone, etc.)
Example: chfn , sudo chfn john
