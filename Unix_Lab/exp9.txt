EXP : 9

grep / sed commands

cmd :
grep -> Searches for a pattern in a file and prints matching lines
Example: grep "hello" file.txt

grep -c -> Counts number of matching lines
Example: grep -c "error" log.txt

grep -o -> Displays only the matched part of the line
Example: grep -o "word" file.txt

grep -v -> Displays lines that do NOT match the pattern
Example: grep -v "test" file.txt

egrep -> Extended grep (supports patterns like OR without escaping)
Example: egrep "pattern1|pattern2" file.txt

wc -l -> Counts number of lines in a file
Example: wc -l file.txt

wc -w -> Counts number of words in a file
Example: wc -w file.txt

wc -c -> Counts number of bytes/characters in a file
Example: wc -c file.txt

wc -m -> Counts number of characters in a file
Example: wc -m file.txt

wc -L -> Displays length of the longest line
Example: wc -L file.txt

sed -> Stream editor used for text processing and replacement
Example: sed 's/old/new/g' file.txt

sed -i -> Edits file in place (modifies original file)
Example: sed -i 's/old/new/g' file.txt

sed -n 'p' -> Prints selected lines from file
Example: sed -n 'p' file.txt

uniq -d -> Displays duplicate lines (used with sorted input)
Example: sort file.txt | uniq -d

sort -> Sorts lines alphabetically
Example: sort file.txt

sort | uniq -d -> Finds duplicate lines
Example: sort file.txt | uniq -d


Script examples for EXP 9:

a) Script using grep to find lines, words, characters in a file:

#! /bin/sh
echo "Enter filename: "
read file
echo "Lines: `wc -l $file`"
echo "Words: `wc -w $file`"
echo "Characters: `wc -c $file`"

b) Script using egrep to display specific type of files:

#! /bin/sh
echo "Enter extension (ex .txt): "
read ext
ls | egrep "$ext$"

c) Script using sed to replace a word in a file:

#! /bin/sh
echo "Enter filename: "
read file
echo "Enter word to replace: "
read old
echo "Enter new word: "
read new
sed -i "s/$old/$new/g" $file
