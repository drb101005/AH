EXP : 9

grep / sed commands

cmd :
grep -> Search for a pattern in file, prints matching lines, ex grep "hello" file.txt
grep -c -> Count number of matching lines, ex grep -c "error" log.txt
grep -o -> Show only the matching part of the line, ex grep -o "word" file.txt
grep -v -> Show lines that do NOT match the pattern, ex grep -v "test" file.txt
egrep -> Extended grep (treats meta-characters as is, no need to escape), ex egrep "pattern1|pattern2" file.txt
wc -l -> Count number of lines in a file, ex wc -l file.txt
wc -w -> Count number of words in a file, ex wc -w file.txt
wc -c -> Count number of bytes/characters in a file, ex wc -c file.txt
wc -m -> Count number of characters in a file, ex wc -m file.txt
wc -L -> Print length of longest line in a file, ex wc -L file.txt
sed -> Stream editor for non-interactive editing, ex sed 's/old/new/g' file.txt
sed -i -> Edit file in place (directly modify file), ex sed -i 's/old/new/g' file.txt
sed -n 'p' -> Print lines from file, ex sed -n 'p' file.txt
uniq -d -> Show only duplicate lines (use with sort), ex sort file.txt | uniq -d
sort -> Sort lines alphabetically, ex sort file.txt
sort | uniq -d -> Find duplicated lines, ex sort file.txt | uniq -d

Script examples for EXP 9:

a) Script using grep to find lines, words, characters in a file:

bash
#! /bin/sh
echo "Enter filename: "
read file
echo "Lines: `wc -l $file`"
echo "Words: `wc -w $file`"
echo "Characters: `wc -c $file`"
b) Script using egrep to display specific type of files:

bash
#! /bin/sh
echo "Enter extension (ex .txt): "
read ext
ls | egrep "$ext$"
c) Script using sed to replace a word in a file:

bash
#! /bin/sh
echo "Enter filename: "
read file
echo "Enter word to replace: "
read old
echo "Enter new word: "
read new
sed -i "s/$old/$new/g" $file
