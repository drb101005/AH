EXP : 2B 
--- VI BASICS ---
vi file_name -> Open/create file

--- MODES ---
i -> Insert before cursor
a -> Insert after cursor
Esc -> Exit insert mode

--- MOVEMENT ---
h -> left
l -> right
j -> down
k -> up
$ -> end of line

--- EDITING ---
x -> delete character
dd -> delete line
dw -> delete word
o -> new line below

--- SAVE & EXIT ---
:w -> save
:wq -> save and quit
:q -> quit
:q! -> force quit
