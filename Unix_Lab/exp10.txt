EXP : 10

awk / perl scripts

cmd :
awk -> Text processing language used for pattern scanning and processing
Example: awk '{print $1}' file.txt

awk '/pattern/' -> Searches for a pattern in input
Example: awk '/hello/' file.txt

awk '{print $NF}' -> Prints last field of each line
Example: awk '{print $NF}' file.txt

awk 'NR%2==0' -> Prints even-numbered lines
Example: awk 'NR%2==0' file.txt

perl -> Perl interpreter to run Perl scripts
Example: perl script.pl

perl -e -> Executes Perl code directly from command line
Example: perl -e 'print "hello\n"'

print -> Prints output in Perl
Example: print "Hello World\n";

my -> Declares a variable in Perl
Example: my $name = "John";

<STDIN> -> Takes input from user in Perl
Example: my $num = <STDIN>;

chomp -> Removes newline character from input
Example: chomp($num);

sort -> Sorts elements (numbers/strings) in Perl
Example: @sorted = sort @array;

use strict -> Enforces strict variable declaration rules in Perl

use warnings -> Enables warning messages in Perl


Script examples for EXP 10:

a) awk script to print all even numbers in a given range:

#! /bin/sh
echo "Enter starting number: "
read start
echo "Enter ending number: "
read end

awk -v s=$start -v e=$end '
BEGIN {
    for(i=s; i<=e; i++) {
        if(i%2==0) {
            print i
        }
    }
}'

Alternative awk one-liner:

awk 'BEGIN{for(i=1;i<=20;i++) if(i%2==0) print i}'

b) awk script to develop Fibonacci series (user input for number of terms):

#! /bin/sh
echo "Enter number of terms: "
read n

awk -v terms=$n '
BEGIN {
    a=0
    b=1
    print "Fibonacci series:"
    for(i=1;i<=terms;i++) {
        print a
        next_term = a + b
        a = b
        b = next_term
    }
}'

c) perl script to check if a number is prime or not:

#! /usr/bin/perl
use strict;
use warnings;

print "Enter a number: ";
my $num = <STDIN>;
chomp($num);

my $is_prime = 1;
if($num < 2) {
    $is_prime = 0;
}
for(my $i=2; $i<=sqrt($num); $i++) {
    if($num % $i == 0) {
        $is_prime = 0;
        last;
    }
}
if($is_prime) {
    print "$num is a prime number\n";
} else {
    print "$num is not a prime number\n";
}

c) Alternative perl script to sort elements of an array:

#! /usr/bin/perl
use strict;
use warnings;

print "Enter numbers separated by space: ";
my $input = <STDIN>;
chomp($input);

my @numbers = split(' ', $input);
my @sorted = sort { $a <=> $b } @numbers;

print "Original array: @numbers\n";
print "Sorted array: @sorted\n";

Perl sort variations:

# Sort numbers in ascending order
@sorted = sort { $a <=> $b } @numbers;

# Sort numbers in descending order
@sorted = sort { $b <=> $a } @numbers;

# Sort strings alphabetically
@sorted = sort @strings;

# Sort strings in reverse order
@sorted = sort { $b cmp $a } @strings;
