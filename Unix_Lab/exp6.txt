Memory Management Commands

cmd :
free -> Displays used and free memory (RAM) and swap usage
Example: free , free -h , free -m , free -g

cat /proc/meminfo -> Displays detailed memory information from system file
Example: cat /proc/meminfo

df -> Displays disk space usage of file systems
Example: df , df -h , df -T , df -m

df -h -> Displays disk space in human-readable format (KB, MB, GB)
Example: df -h

df -T -> Displays disk space along with file system type
Example: df -T

du -> Displays disk usage of files and directories
Example: du , du -h , du -sh , du -sh /home

du -sh -> Displays total size of a directory (summarized, human-readable)
Example: du -sh Documents/ , du -sh *

vmstat -> Displays system performance (memory, processes, CPU, paging)
Example: vmstat , vmstat 5 , vmstat -s
