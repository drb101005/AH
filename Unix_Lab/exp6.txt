Memory Management Commands
cmd :
free -> Display used and unused memory (RAM) and swap memory usage
Example: free , free -h , free -m , free -g

cat /proc/meminfo -> Display detailed system memory usage information from virtual file system
Example: cat /proc/meminfo

df -> Display disk space used in the file system (disk filesystem)
Example: df , df -h , df -T , df -m

df -h -> Display disk space usage in human-readable form (KB, MB, GB)
Example: df -h

df -T -> Display disk space usage along with file system type
Example: df -T

du -> Display disk usage of files and directories (Disk Usage)
Example: du , du -h , du -sh , du -sh /home

du -sh -> Display total size of a directory in human-readable format (summarize)
*Example: du -sh Documents/ , du -sh * *

vmstat -> Display system performance information (processes, memory, paging, CPU)
*Example: vmstat , vmstat 5 , vmstat -s*
