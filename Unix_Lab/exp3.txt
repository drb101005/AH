File System Management Commands

cmd :
ls -> Lists directory contents (files and folders)
cd -> Changes current directory
Example: cd /home

mkdir -> Creates a new directory
Example: mkdir myfolder

rmdir -> Removes an empty directory
Example: rmdir myfolder

mv -> Moves or renames a file/directory
Example: mv file1.txt file2.txt

touch -> Creates a new empty file
Example: touch myfile.txt

> -> Redirects output to create/overwrite a file
Example: echo "Hello" > newfile.txt

echo -> Prints text to terminal (also used with redirection)
Example: echo "Hello World"

rm -> Deletes file permanently
Example: rm myfile.txt

cp -> Copies file or directory
Example: cp file1.txt file2.txt

| -> Pipe - sends output of one command as input to another
Example: ls -la | grep ".txt" , cat file.txt | wc

grep -> Searches for a specific pattern/string in a file
Example: grep "hello" file.txt

sort -> Sorts file contents (alphabetically/numerically)
Example: sort names.txt , sort -r names.txt , sort -n numbers.txt

tr -> Translates or replaces characters (e.g., lowercase to uppercase)
Example: echo "hello" | tr 'a-z' 'A-Z' , cat file.txt | tr ' ' '\n'

head -> Displays first 10 lines of a file (default)
Example: head file.txt , head -n 20 file.txt

tail -> Displays last 10 lines of a file (default)
Example: tail file.txt , tail -n 15 file.txt , tail -f logfile.txt

wc -> Counts lines, words, and characters in a file
Example: wc file.txt , wc -l file.txt , wc -w file.txt , wc -c file.txt

EXP : 3C

File System Management: Compression and Archiving Commands

cmd :
zip -> Compresses files into a .zip archive
Example: zip myarchive.zip file1.txt file2.txt

unzip -> Extracts files from a .zip archive
Example: unzip myarchive.zip

gzip -> Compresses file and replaces it with .gz file
Example: gzip myfile.txt , gzip -c myfile.txt > myfile.txt.gz

gunzip -> Decompresses .gz file and restores original file
Example: gunzip myfile.txt.gz

tar -> Archiving utility to create/extract archive files
Example: tar -cvf myarchive.tar *.txt

tar -cvf -> Creates uncompressed .tar archive
Example: tar -cvf myarchive.tar file1.txt file2.txt , tar -cvf backup.tar /home/user/*

tar -xvf -> Extracts files from .tar archive
Example: tar -xvf myarchive.tar , tar -xvf myarchive.tar -C /target/folder

tar -cvzf -> Creates gzip compressed tar archive (.tar.gz)
Example: tar -cvzf myarchive.tar.gz *.c , tar -cvzf backup.tar.gz /home/user/

tar -xvzf -> Extracts gzip compressed tar archive (.tar.gz)
Example: tar -xvzf myarchive.tar.gz

tar -j -> Uses bzip2 compression (.tar.bz2)
Example: tar -cvjf myarchive.tar.bz2 *.txt , tar -xvjf myarchive.tar.bz2

tar -t -> Lists contents of archive file
Example: tar -tvf myarchive.tar , tar -tvf myarchive.tar.gz

tar -u -> Updates archive by adding new files
Example: tar -uvf myarchive.tar newfile.txt

tar -A -> Concatenates multiple archive files
Example: tar -Af archive1.tar archive2.tar

tar -r -> Appends files/directories to existing .tar file
Example: tar -rvf myarchive.tar newfile.txt

tar -W -> Verifies integrity of archive file
Example: tar -Wvf myarchive.tar
